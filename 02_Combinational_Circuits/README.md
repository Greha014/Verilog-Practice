# Combinational Circuits

This folder contains Verilog HDL implementations and testbenches for common combinational circuits.

## Implemented Modules

- Half Adder
- Full Adder
- Full Adder using Half Adder
- 2:1 Multiplexer
- 4:1 Multiplexer
- 1:2 Decoder
- 3:8 Decoder
- 1:4 DEMUX
- 1:8 DEMUX
- 4:2 Encoder
- 8:3 Encoder
- Priority Encoder
- 4-bit RCA
- 4-bit CLA
