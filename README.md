# Verilog Practice

- AND Gate
- OR Gate
- NAND Gate
- NOR Gate
- XOR Gate
- XNOR Gate
- NOT Gate

### Combinational Circuits
- Multiplexers
- Demultiplexers
- Encoders
- Decoders

### Sequential Circuits
- D Flip-Flop
- JK Flip-Flop
- T Flip-Flop

### Counters
- Up Counter
- Down Counter
- Mod-N Counter

### Shift Registers
- SISO
- SIPO
- PISO
- PIPO

### Finite State Machines
- Sequence Detector
- Traffic Light Controller

## Tools Used
- Verilog HDL
- ModelSim / QuestaSim
- GTKWave

Build strong RTL Design fundamentals for Front-End VLSI and FPGA Design.
